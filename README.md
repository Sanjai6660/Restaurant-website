# 🍽️ Restaurant Website

A modern and responsive **Restaurant Website** built using **HTML, CSS, and JavaScript**.  
The project focuses on delivering an elegant UI, smooth navigation, and a mobile-friendly experience.

This website showcases a restaurant's menu, story, reservation system, and contact information in a clean and interactive layout.

---

## 🚀 Features

- Responsive design for **desktop, tablet, and mobile**
- Elegant **homepage layout**
- Interactive **food menu section**
- **Table reservation form**
- **About / Restaurant Story** section
- **Contact section with location map**
- Smooth navigation and user-friendly layout

---

## 🛠️ Technologies Used

- **HTML5**
- **CSS3**
- **JavaScript (ES6)**
- **Responsive Web Design**
